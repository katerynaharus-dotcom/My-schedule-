# 📅 Мій робочий графік

Застосунок для відображення робочого графіку з можливістю поділитись посиланням.

## Як розгорнути на Vercel

### Крок 1 — Завантажити на GitHub
1. Зайди на [github.com](https://github.com) → натисни **"New repository"**
2. Назви репозиторій: `my-schedule`
3. Натисни **"Create repository"**
4. Завантаж всі файли з цієї папки (кнопка **"uploading an existing file"**)

### Крок 2 — Розгорнути на Vercel
1. Зайди на [vercel.com](https://vercel.com) → **"Add New Project"**
2. Вибери репозиторій `my-schedule`
3. Натисни **"Deploy"** — Vercel сам все зробить!
4. Через ~1 хвилину отримаєш посилання типу `my-schedule.vercel.app`

## Як користуватись

- **Твоє посилання** (для редагування): `my-schedule.vercel.app`
- **Посилання для колег** (тільки перегляд): `my-schedule.vercel.app?view=1`

Кнопка 🔗 **Поділитись** автоматично генерує посилання для колег.
